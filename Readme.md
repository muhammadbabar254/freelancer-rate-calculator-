# 💰 Freelancer Rate Calculator

A smart and simple Freelancer Rate Calculator built with HTML, CSS, and JavaScript.  
It helps freelancers calculate their hourly, daily, and monthly rates based on income goals.

---

## ✨ Features

- 💵 Monthly income target input
- 📅 Working days configuration
- ⏱️ Working hours per day
- 📊 Auto calculation of:
  - Hourly rate
  - Daily rate
  - Monthly rate
- 🎨 Clean modern UI
- ⚡ Instant calculation (no reload)

---

## 🚀 How to Use

1. Open `index.html` in browser
2. Enter your monthly income goal
3. Set working days per month
4. Set working hours per day
5. Click **Calculate Rate**
6. View your pricing instantly
